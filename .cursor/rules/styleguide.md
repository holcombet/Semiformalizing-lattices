# Markdown styleguide

Cursor applies Markdown authoring rules from `styleguide.mdc` in this folder (YAML `globs` and `description`). Edit `styleguide.mdc` to change the policy.

The **shared emphasis norms** in that file are kept in sync with `chapman-alignment-faking/.cursor/rules/styleguide.mdc`; update **both** when you change those norms.
