**lemma_5_3_i** If $x$ is an atom, then $x$ is join-irreducible

*Proof:*

> Ass: $x$ is an atom.
> Have: $\bot < x$
> Have: $\bot < y \le x \ \Rightarrow y=x$.
> Show: $y\le x\ \Rightarrow y=x$ 
> > ...
> 
> Show: $b\sqcup c = x \ \Rightarrow b=x \vee c=x$.
> > Case: $b = x$
> > Have: $b = x$ QED
>
> > Case $b\not=x$
> > Have: $b\le x$
> > Have: $b=\bot$
> > Have: $b\sqcup c = c = x$ QED
> 
> QED
